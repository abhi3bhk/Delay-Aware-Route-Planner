# Delay-Aware-Route-Planner
Delay-Aware Route Planner using shortest path algorithms (DAA project)

## Description
This project analyzes shortest path algorithms under dynamic conditions using a graph-based model.

## Features
- Graph using adjacency list
- Dijkstra algorithm
- Bellman-Ford algorithm
- Delay simulation
- Route comparison

## Algorithms
- Dijkstra (Greedy)
- Bellman-Ford (Dynamic Programming style)
